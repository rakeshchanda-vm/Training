Basic Concepts
==============

Foundational concepts every Andromeda user needs to understand. These guides cover the core building blocks in logical dependency order.

**Learning Path:**

1. :doc:`models` - Working with language models and model configuration
2. :doc:`agents` - Creating and configuring intelligent agents
3. :doc:`tools` - Tool system, built-ins, and custom tools
4. :doc:`supervisors` - Orchestrator agents and task routing
5. :doc:`teams` - Multi-agent coordination and team patterns
6. :doc:`workspace-agents` - Long-horizon agents bound to an isolated workspace
7. :doc:`workflow-builder` - Declarative workflow construction
8. :doc:`configuration` - Configuration management and YAML setup
